GradeGenius is a simple web application that allows students to calculate their GPA by entering course names, grades, and credit weights. It is a lightweight front-end project built using HTML, CSS, and JavaScript.

Features

Add and remove courses

Automatic GPA calculation

Uses a standard 4.0 GPA scale

Clean and easy-to-use interface

Technologies Used

HTML for structure

CSS for styling

JavaScript for logic and dynamic updates

How It Works

Enter a course name, grade percentage, and credit weight.

Click "Add Course" to add it to the table.

Click "Calculate GPA" to see your weighted GPA.

Courses can be removed individually.
